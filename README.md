# STUDENT-DATA-MGMT-PROJECT
This Computer Science project is based on the Class 12th CBSE NCERT curriculum and focuses on managing student records efficiently using programming and database concepts.  The system is designed to store, retrieve, update, and manage student information in a structured manner using MySQL database, instead of manual record keeping.

# Requirements & Setup

To run this Project, make sure the following requirements are fulfilled:

# System Requirements

Operating System: Windows / Linux / macOS
Minimum 4 GB RAM (recommended)

# Software Requirements

Python (Latest Version Recommended)
MySQL Server
MySQL Connector for Python
